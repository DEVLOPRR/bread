Bread

Install, update and remove AppImage from GitHub using your CLI

Features

  • Install From GitHub Release
  • Update With Ease

Installation

With Curl

sudo curl -L https://github.com/DEVLOPRR/bread/releases/download/v0.2.2/bread-0.2.2-x86_64.AppImage -o /usr/local/bin/bread && sudo chmod +x /usr/local/bin/bread

With Wget

sudo wget -O /usr/local/bin/bread https://github.com/DEVLOPRR/bread/releases/download/v0.2.2/bread-0.2.2-x86_64.AppImage && sudo chmod +x /usr/local/bin/bread

Usage

Install a application

bread install user/repo

Remove a application

bread remove user/repo

Update a application

bread update user/repo

List all the installed application

bread list
NOTE

if you are trying to install/remove/update a application where the user and repo are same, like bread install libresprite/libresprite can be written as bread install libresprite

Building from source

Make Sure You Have Go version 1.17.x & AppImage Builder Installed.

Get The Repository Via Git

git clone https://github.com/DEVLOPRR/bread

Go Inside The Source Code Directory & Get All The Dependencies

cd bread
go mod tidy

Make The Build Script Executable And Run It

chmod +x ./make
./make

And To Build The AppImage Run

./make appimage
